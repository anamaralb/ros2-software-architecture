# ROS2 Software Architecture
This repository contains several ROS2-based robotics projects developed during a Software Architecture course in my Robotics Engineering degree.

The objective was to design modular robotic software using ROS2, Behavior Trees and C++.

## Projects

### Obstacle Avoidance
ROS2 node for obstacle avoidance using laser sensor data.

### Receptionist Behavior Tree
Behavior Tree based robot interaction system.

### Seek and Capture
Robot perception and control architecture using Behavior Trees.

## Technologies

- ROS2
- C++
- BehaviorTree.CPP
- Linux
- CMake
