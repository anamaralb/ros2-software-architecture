# Seek and Capture (ROS2)

ROS2 package implementing a perception and control architecture for a robot using Behavior Trees.

The robot searches for a target, navigates towards it, and executes actions depending on the environment state.

## Features

- Behavior Tree based control
- Perception + action architecture
- ROS2 node communication
- Modular design

## Technologies

- ROS2
- C++
- BehaviorTree.CPP
- rclcpp

## Run

Use the launch file included in the package.
