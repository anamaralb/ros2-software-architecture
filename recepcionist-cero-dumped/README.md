# Receptionist Behavior Tree (ROS2)

ROS2 package implementing a robot interaction behavior using Behavior Trees.

The system manages robot actions through a modular architecture, allowing the robot to react to events and execute tasks using a Behavior Tree structure.

## Features

- Behavior Tree control
- Modular node architecture
- ROS2 communication
- Event-driven execution

## Technologies

- ROS2
- C++
- BehaviorTree.CPP
- rclcpp

## Run

Use the corresponding launch file inside the package.
